Welcome to RISING SUN BIKE RACING GAME!!!

MAIN MENU:

	To start a new game, use "NEW GAME" option.

	To open the instruction, use "INSTRUCTION" option.
	
	To exit the game, use "EXIT" option.

GAMEPLAY:

	To move, you need to have positive speed.

	To increase it, use arrow right key. To decrease - arrow left key.

	You will need to avoid obstacles. To jump over them, use arrow up key.

	As soon as you collide with an obstacle or finish the level, the game is over.

	You can begin a new one by using the same "NEW GAME" option in the main menu.
